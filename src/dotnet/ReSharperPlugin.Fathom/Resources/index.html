What Fathom Does

Code Structure Analysis

Describe the structure of any source file in your solution: classes, methods, fields, inheritance, and more. Works with C++ and Blueprints.

Code Inspection

Run ReSharper-powered code inspections on demand. Catch issues, warnings, and suggestions for any file via a simple HTTP call.

C++ Class Discovery

List and search game C++ classes with their header/source file pairs and base class info. Filter by name or base class hierarchy.

Blueprint Auditing

Audit Blueprint classes: node counts, complexity, variable usage, and event graphs. Identify Blueprints that may need refactoring or C++ migration.

Asset Intelligence

Search and browse UAssets, inspect metadata, and trace dependency and referencer graphs across your Unreal project.

MCP Integration

Expose all capabilities as MCP tools for AI coding assistants. Works with Claude Code, Cursor, VS Code Copilot, and other MCP-compatible clients.

API Endpoints

Method Path Description
GET /files List all source files in the solution (JSON only)
GET /classes List game C++ classes with header/source pairs and base class
GET /describe_code?file=path Structural description of source files
GET /inspect?file=path Run code inspection on source files
GET /blueprints?class=name List Blueprint classes deriving from a C++ class UE5
GET /bp?file=/Game/Path Blueprint composite info (audit + deps + refs) UE5
GET /blueprint-audit Get Blueprint audit data UE5
GET /uassets?search=term Search or browse UAssets UE5
GET /uassets/show?package=path Asset detail: metadata, size, tags, deps UE5
GET /asset-refs/dependencies Asset dependency graph UE5
GET /asset-refs/referencers Asset referencer graph UE5
GET /symbols Search code symbols across the solution
GET /health Server and solution status (JSON)
POST /mcp MCP Streamable HTTP endpoint (JSON-RPC)

All GET endpoints default to markdown. Add &format=json for JSON output.

MCP Configuration

Connect AI coding assistants to Fathom via the Model Context Protocol. Add this to your MCP client configuration:

{
  "mcpServers": {
    "fathom": {
      "url": "http://localhost:{{PORT}}/mcp"
    }
  }
}

Fathom auto-generates .mcp.json in your solution root for compatible clients.
